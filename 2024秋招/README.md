1、加锁以后，抛了异常，导致锁没有释放，怎么解决

2、写一个函数判断大端、小端

小端：数据低位保存在低地址    
大端：数据低位保存在高地址    
定义一个 int 值为 1，然后将地址强制转化给一个 char* 就可以得到这个 int 的前四位    
如果这个 char* 的值是 1，说明是小端，否则是大端    
```c++
solve() {
    int i = 1;
    char c = *((char*) &i);
    if (c) cout << "小端";
    else cout << "大端";
}
```
