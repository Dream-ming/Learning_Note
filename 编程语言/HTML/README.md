```html
<html>
    <head>
        <title>test</title>
    </head>
    <body>
        <table>
            <tr>
                <td>
                    <a href="网络地址或资源文件">name</a>
                </td>
                <td></td>
            </tr>
            <tr>
                <td></td>
                <td></td>
            </tr>
        </table>
    </body>
</html>
```

* head : 头部标签，在浏览器最上面那块
* body : 网页
* table : 一块内容
* tr : 行
* td : 列
* a : 跳转
